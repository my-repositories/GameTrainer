# GameTrainer
Powered by C++17 & Lua-5.3.4

## Support Games:
- Dead Island
- Dead Island: Riptide
- Killing Floor
- Killing Floor 2
- Left 4 Dead 2

## Install
- CMake: https://cmake.org/download/
- Visual Studio 2019: https://visualstudio.microsoft.com/ru/vs/
- (Optional) Lua: https://www.lua.org/download.html

## Build
`make run`
